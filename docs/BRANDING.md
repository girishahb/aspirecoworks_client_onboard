# ASPIRE COWORKS Branding

## Current Brand
- **Company name:** ASPIRE COWORKS
- **Font:** Arial Bold (#134b7f)
- **Primary color:** HEX #134b7f

## Logo
The app uses `frontend/public/logo.svg` as the company logo. To use your own logo:

1. Place your logo file in `frontend/public/` as `logo.svg` (or `logo.png`)
2. If using PNG: update the Logo component to use `logoSrc="/logo.png"` where the Logo is rendered (or set the default in `frontend/src/components/Logo.tsx`)

## Replacing the Logo
Replace `frontend/public/logo.svg` with your actual logo. Supported formats: SVG (recommended for crisp display) or PNG.
